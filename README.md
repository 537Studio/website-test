开发完毕后会进行发布

### 目标

重构 537 工作室的官方网站，尽力达成以下目标

- UI 合理(使用 Next.js & React 相关的 UI 库和 Next.js，让 UI 尽力简洁合理)
- 性能优化(抛弃传统的 MPA 模式，使用 SPA&SSR 模式，更好的进行 SSR 和 SSG 渲染)
- JS 控制(不使用传统的 HTML 多次复制粘贴，使用 js 函数等进行控制)

### 技术栈

- Next.js (App Router)
- Zustand
- Framer motion
- Tailwind CSS
